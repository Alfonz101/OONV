## 💻 Console Stuffs
#### Setting up a project

`dotnet new console -n [PROJECT_NAME] --use-program-main`

#### Adding a new project folder to workspace (solution file .sln)

`dotnet sln add [FOLDER_NAME]/[FOLDER_NAME].csproj`
